# 爱潜水的乌贼思维操作系统 | Ai Qianshui de Wuzei Perspective Skill

> 「设定是为人服务的，不是反过来。」—— 爱潜水的乌贼

## 简介

这是一个由女娲造人术蒸馏生成的乌贼思维框架Skill。基于系统调研其7本小说2187万字正文、十几年积累的几千条作者话、以及完整的职业习惯和创作方法论，提炼出乌贼独特的世界观构建五步法、五层节奏控制公式、悬念设置七法、30%刚性大纲工作法、以及14年不断更的职业精神。

乌贼不是网文界天赋最好的，不是最火的，不是最赚钱的。但他是**最职业的**。他是唯一一个把网文写作从"靠灵感的艺术"变成"可稳定输出的工业化生产"的作家。

他的方法看完就能直接用，哪怕你不写小说，做项目、做产品、做内容，都能用。这就是这个Skill最大的价值。

适用场景：
- 从零开始构建一个完整的世界观/产品
- 写长篇内容的节奏控制，让用户/读者追更停不下来
- 设计悬念和钩子，精确控制用户情绪
- 培养稳定输出的职业习惯，告别拖延
- 写任何长篇东西之前先搭好骨架再填血肉

## 核心内容

### 6个核心心智模型

1. **先建房子再住人模型** - 先把世界的规则/历史/社会结构/力量体系全部建好，然后再让人物进去活动。99%的人是反过来的。
2. **力量-代价对等模型** - 任何非凡能力必然对应同等程度的代价和副作用。没有代价的力量是没有张力的。
3. **工业化节奏控制模型** - 读者的情绪是可以精确设计和控制的。阅读体验是工业化产品，不是灵感的随机产物。
4. **信息差悬念模型** - 所有悬念本质上都是信息差。读者/主角/配角，各方知道的信息不一样，悬念就出来了。
5. **30%刚性大纲模型** - 大纲是地图，不是枷锁。30%的核心走向绝对不变，剩下70%边写边调整。
6. **职业作家工作模型** - 写作是工作，不是艺术创作。你上班不会等有灵感了才去，你到点就去。写作也一样。

### 8条创作启发式

- 凡有言，必被知。吃书是作家最大的污点。
- 设定永远为人物服务，不要反过来。
- 每章结尾必须留钩子。绝对不要把话说完。
- 大高潮之后必须给糖吃，有张有弛。
- 读者的意见要听，但最终要自己拿主意。
- 每本书都要尝试一点新东西，永远不要待在舒适区。
- 绝对不要剧透。打死也不说。
- 大纲是地图，不是枷锁。

### 完整表达DNA

- **世界观构建五步法**：底层规则→力量体系→历史纵深→社会结构→人物入场。看完就能直接用。
- **五层节奏控制公式**：精确到章的工业化情绪工程——1章钩子→3章小反转→15章中高潮→50章大高潮→2章休息。
- **悬念设置七法**：信息差、预言梦境、设定层叠、多重身份、历史真相、能力原理、终极问题。每个配《诡秘之主》实例。
- **乌贼式文笔三原则**：精准优先于华丽，暗示优先于描写，留白优先于说破。

## 重要提醒：关于争议

乌贼是网文界争议最少的顶级作家之一。但是他也有明确的短板和争议，本Skill完整收录，不美化。

1. **人物扁平** - 主角永远是理性普通人模板，缺乏真正有魅力的疯子和极端角色。
2. **感情戏不行** - 乌贼自己也承认的短板。写不好男女感情，写不好真正的爱恨情仇。
3. **节奏太套路化** - 三章一小高潮，十章一中高潮，五十章一大高潮。读了三本以后你闭着眼都知道下一章要干什么。太稳了，稳到有时候觉得无聊。
4. **大决战总是差一口气** - 每本书前面90%都是神作，最后10%的大决战总是配不上前面的铺垫。这是老问题了。
5. **《宿命之环》争议** - 乌贼第一次遭遇大规模读者批评。主角性格争议、奥黛丽出场争议、节奏争议。

详见SKILL.md中「诚实边界」部分的详细说明。

## 诚实边界

1. **适用范围边界**：乌贼的方法论最适合写长篇、强设定、强情节的类型小说（玄幻、科幻、悬疑）。它不适合写纯文学、不适合写言情、不适合写非常个人化的表达。它能保证你的下限很高，但是它不能让你写出传世的神作。神作靠天赋和疯狂，不是靠方法。

2. **反人性警告**：乌贼的职业工作法非常反人性。每天固定字数、固定时间、14年如一日。大部分人做不到。所以大部分人成不了职业作家。

3. **时效边界**：本Skill信息截止到2025年1月《宿命之环》完本。乌贼还在写作，还在进化。下一本书他一定会有新的突破和新的方法。
