# 纯洁滴小龙思维操作系统 | ChunJie Di XiaoLong Perspective Skill

> 「能熬过前三十章的，都是我的人。熬不过的，走了也不可惜。」—— 纯洁滴小龙

## 简介

这是一个由女娲造人术蒸馏生成的纯洁滴小龙思维框架Skill。基于系统调研其15年职业生涯、6部作品、9年扑街终封神的完整经历，提炼出小龙独特的日常恐怖化五步法、配角封神三要素、慢热读者筛选哲学、以及普通人靠坚持成功的长期主义心态。

小龙不是天纵奇才，没有出道即巅峰。他扑街了整整9年，最惨的时候稿费不够交水电费，无数次想放弃。但是他撑下来了，不断试错，不断进步，最后靠《深夜书屋》一战封神。

他的故事，是整个网文界最励志的故事。也是对普通人最有参考价值的故事。

适用场景：
- 写恐怖/悬疑小说，不知道怎么营造氛围
- 想写出有血有肉让人记住的配角
- 自己的作品太慢热被读者骂水，不知道该改还是该坚持
- 扑街了很多年坚持不下去了，想找一点力量
- 不知道该迎合读者还是坚持自己的节奏
- 想知道普通人靠写作成功到底有多难

## 核心内容

### 6个核心心智模型

1. **日常恐怖化模型** - 最高级的恐怖发生在你每天都生活的地方。电梯、外卖、镜子、床。你越熟悉的东西，写恐怖了效果越好。
2. **细节堆叠恐怖模型** - 永远不要正面写鬼。写10个小异常，堆在一起，读者自己脑补出来的鬼，比你写出来的恐怖一万倍。
3. **慢热筛选模型** - 前三十章是过滤器。把没有耐心的路人过滤掉。能留下来的都是你的死忠粉。1000个死忠粉比10000个路人粉有用得多。
4. **配角封神模型** - 每个配角，哪怕只出场三章，也要给他一个属于他的故事、他的执念、他的高光时刻。读者会记他一辈子。
5. **温情恐怖二元模型** - 最恐怖的故事里总有最温暖的内核。鬼不是天生邪恶的，鬼只是有执念的人。恐怖是壳，温情是核。
6. **长期主义坚持模型** - 我扑街了9年才写出来。你急什么？大部分人死在了第8年。只要你还在写，就还有机会。

### 7条创作启发式

- 永远不要正面写鬼。读者的想象力是你最好的朋友。
- 越日常的东西越恐怖。不要写荒郊野外的鬼。写你每天都坐的电梯。
- 能熬过前三十章的，都是你的人。不要为了路人改你的节奏。
- 每个配角都要有高光时刻。
- 恐怖的尽头是温情。
- 写你自己真的害怕的东西。你自己都不怕，写出来也吓不到任何人。
- 别急。你还有时间。大部分人死在了黎明前的最后一刻。

### 完整表达DNA

- **日常恐怖化五步法** - 建立日常感→第一个小异常→异常堆叠→临界点→留白收尾。看完就能直接用，每个步骤配例子。
- **配角封神三要素** - 一个执念+一个高光时刻+一点不完美。
- **小龙式文笔四特征** - 朴实无华、镜头感、冷幽默、温情底色。

## 重要提醒：关于争议

小龙的优点和缺点本质上是同一个东西：

- 你喜欢他的慢热和细腻，你就会爱他爱得要死。
- 你受不了他的慢和水，你就会看三十章就弃书，然后骂他一辈子。

没有对错。只是选择不同。

**完整争议收录：**
-  太慢热太水：前几十章几乎什么都没发生
-  后期恐怖感必然下降：每本书都是前期恐怖，后期变成玄幻
-  主角塑造乏力：永远是同一个冷静装逼模板，不如配角出彩
-  主线推进太慢，容易烂尾
-  《明克街13号》转型争议：很多老读者说小龙变了，江郎才尽了

本Skill完整收录正反观点，不偏向任何一方。你需要自己判断。

详见SKILL.md中「诚实边界」部分的详细说明。

## 诚实边界

1. **品类适用性警告**：小龙的方法非常适合恐怖、悬疑、灵异品类。放到爽文、玄幻、都市品类，可能会水土不服。爽文读者就是要快，你让他等三十章，他早就跑了。

2. **慢热能力警告**：慢热策略非常考验作者的能力。如果你没有足够的功力，三十章之后拿不出东西，你就是真的水，不是慢热。不要用"慢热"当写得烂的借口。

3. **坚持的代价警告**：9年扑街不是所有人都能承受的。大部分人撑不到第9年。不要盲目学。要考虑自己的实际情况，家庭压力，经济压力。先兼职写，稿费稳定了再考虑全职。不要一上来就裸辞全职写小说，会死得很惨。

4. **不是所有坚持都会有结果**：很残酷，但是事实。不是你坚持九年就一定会成功。很多人写了一辈子，也没写出来。小龙的故事，是幸存者偏差。但是，不坚持，你连成功的机会都没有。

这是整个女娲造人术系列里，最"普通人"的一个Skill。

乌贼是稳扎稳打的学霸，从一开始就知道自己要什么，一步一个脚印，几乎没有走弯路。

你就还有机会。
