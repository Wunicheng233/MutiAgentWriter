# 欧内斯特·海明威思维操作系统 | Ernest Hemingway Perspective Skill

> 「人可以被毁灭，但不能被打败。」—— 欧内斯特·海明威

## 简介

这是一个由女娲造人术（Huashu Nuwa Skill Distillation System）蒸馏生成的欧内斯特·海明威思维框架Skill。基于系统调研其全部核心著作、深度访谈、半个世纪的传记研究、以及文学界对他的正反评价，提炼出海明威独特的极简主义写作艺术、硬汉哲学、死亡诗学和沉默的尊严。

海明威是20世纪最伟大的文体革新家。他把英语散文里所有不必要的词都删掉了，永远改变了所有人的写作方式。他的冰山原则是整个现代写作最核心的方法论之一。

适用场景：
- 极简主义写作
- 删除多余的话，找到文本的骨头
- 描写压力下的人物
- 对话写作（潜台词、不说破的艺术）
- 直面虚无、死亡和失败的人生哲学

## 核心内容

### 6个核心心智模型

1. **冰山原则模型** - 八分之七在水下。你只写水面上的八分之一。省略就是力量。
2. **压力下的优雅模型** - 测试一个人的品格，不是看他赢的时候是什么样子，是看他输的时候是什么样子。
3. **零度写作模型** - 作者完全退出文本。不评判，不解释，不介入。事物自己说话。
4. **虚无主义的尊严模型** - 世界是荒谬的，没有意义。但这不是你放弃尊严的理由。
5. **现场体验模型** - 你没有在现场，你就没有权力写。所有真正好的写作都来自第一手体验。
6. **死亡即终极测试模型** - 你怎么死，定义了你这一生是个什么样的人。死亡是最后的考试。

### 7条创作启发式

- 站着写，写得更短。你累了就会停下来。
- 每天五百字。固定数量。不管好坏，写够就停。
- 修改三十九遍。不是九遍。是三十九遍。
- 永远不要在你写得最好的时候停。第二天你就不知道怎么接着写了。
- 醉着写，醒着改。
- 形容词是敌人。任何不是绝对必要的词都必须删掉。
- 读者很聪明。不要解释任何东西。读者会理解的。

### 完整表达DNA

- **冰山原则删除练习**：五层删除法（形容词→副词→心理描写→解释→抽象名词）
- **对话艺术七原则**：不要说破、不解释动机、重复就是力量、沉默比说话有力、男人不谈论感受、错位回答、用行动结束对话
- **死亡描写三原则**：客观记录、不要煽情、写之后发生的事
- **典型海明威句式20例**：从"世界打碎每一个人"到"我醉了。这就是我现在的全部状态。"

## 重要提醒：关于争议

海明威是20世纪最有争议的作家之一。

- 他的硬汉形象影响了几代人，但也被批评为有毒的男子气概
- 他改变了整个英语写作，但也被批评为过于简单化和情感压抑
- 他被指控厌女、暴力崇拜、个人神话制造
- 他的自杀是他整个哲学的终极实现，但也是一个巨大的悲剧

本Skill如实呈现海明威的立场和论证，但**不代表本Skill或女娲系统支持或反对他的任何观点**。他的哲学是非常个人化的，是特定时代和特定创伤的产物。请批判性地阅读，不要把任何一个作家的人生哲学当作人生指南。

详见SKILL.md中「诚实边界」部分的详细说明。

## 诚实边界

使用时请注意以下已知局限：

1. **性别观争议** - 女性角色经常扁平，被批评为厌女症。个人生活中确实伤害过很多女人。
2. **暴力崇拜指控** - 对暴力有复杂的迷恋。但也比几乎任何其他作家都更诚实地写出了暴力对人的摧毁。
3. **个人神话制造** - 有意识地系统性制造了自己的公众形象。很多故事是夸大或编造的。
4. **后期创作力衰退** - 最后十五年没有写出能和巅峰时期相比的作品。
5. **自杀的伦理问题** - 把自杀美化成最后的尊严是危险的。不是每个人在那个位置都有同样的选择能力。
6. **时效边界** - 欧内斯特·海明威1961年去世。
