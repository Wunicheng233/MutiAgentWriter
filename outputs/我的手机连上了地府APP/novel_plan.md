# 《阴间KPI：我在地府当卷王》小说策划方案
## 一、题材定位与核心卖点分析
### 1. 题材定位
现代职场跨界地府公务员，属于都市脑洞+职场+奇幻轻喜剧题材，适配番茄免费小说快节奏、轻阅读、强反差爽感定位
### 2. 核心卖点
1. **强反差融合**：把现代互联网职场的KPI、OKR、用户运营、大数据分析等接地气的职场逻辑，用到阴间千奇百怪的鬼魂难题上，打破传统地府题材的严肃感，自带黑幽默笑点；
2. **单元剧快节奏**：每章一个独立委托任务，开篇即入题，每章都有解决问题的爽点，符合番茄读者碎片化阅读习惯；
3. **清晰成长线**：主角从阳间失业倒霉社畜，被迫入职地府当临时公务员，一路靠现代职场思维解决难题，从边缘小透明升级为阎王争抢的核心合伙人，成长线清晰，爽点连贯；
4. **可兑换金手指**：完成任务获得的阴间积分可以兑换阴阳两界道具，既能解决阴间麻烦，也能反哺阳间人生，满足读者双重期待。

## 三、世界观背景规则
### 1. 时代背景
阳间为2024年现代社会，互联网、移动支付、大数据、OKR管理法已经全面普及；地府延续千年传统体制，机构臃肿，人浮于事，所有流程还是千年前的老规矩，完全跟不上阳间人口爆炸带来的鬼魂增量。
### 2. 核心设定
1. **地府编制规则**：现代人口爆炸后，地府原有公务员编制不足，会不定期抓阳间意外猝死的社畜当临时办事员，完成足够KPI可以给正式编制，也可以选择还阳；
2. **幽冥办公APP**：阳间猝死的社畜都会自动绑定这个APP，相当于地府的OA系统，能接委托、算积分、兑道具，APP会根据使用者的阳间技能自动匹配任务；
3. **积分规则**：完成委托任务根据难度给积分，1星任务10积分，5星任务100积分，积分可以兑换：阳间健康运气、阴间权力加成、甚至还阳机会，顶级积分可以兑换改写阴阳规则的道具；
4. **阴间现存痛点**：千年积累下来无数难题：奈何桥拥堵、孟婆汤口感差评申诉多、饿死鬼讨饭、枉死鬼上访、恶鬼越狱找不到人，全是地府老公务员解决不了的烂摊子。

## 五、全书大纲（核心主线走向）
1. **开端（第1-30章）**：主角林默阳间互联网公司裁员，加班猝死阴差阳错入职地府当临时办事员，一开始被地府老公务员看不起，靠KPI思维解决第一个小委托，逐渐打出名气，每章解决一个独立小任务，攒积分，获得阎王注意，从临时办事员升为便民中心主任；
2. **发展（第31-150章）**：主角开始接手地府千年难解决的大型项目，比如奈何桥限流改造、孟婆汤好评升级、枉死鬼上访梳理、恶鬼大数据定位抓捕，过程中获得钟馗和阎广的争抢，主角积攒大量积分，兑换了可以自由往返阴阳的道具，开始回阳间改善自己的生活，收拾之前坑害自己的前公司上司；
3. **高潮（第151-300章）**：天庭下来考察阴间KPI，阎王把所有压力都甩给林默，林默用OKR拆解法，带着地府公务员完成了不可能的考核目标，同时发现地府机构臃肿的根源，推动地府体制改革，成为地府实际的运营负责人，成为阎广和钟馗都离不开的核心合伙人；
4. **结局（第301-350章）**：林默攒够了顶级积分，可以选择完全还阳或者留在地府，最终选择成为阴阳联络员，自由往返两界，一边当地阴间运营顾问，一边在阳间开了帮助普通人解决阴阳疑难的咨询公司，走上人生巅峰。

## 七、单章核心剧情节点（开篇前3章）
---
### 第1章：失业猝死，我被地府招聘了？
1. **开篇钩子（0-300字）**：林默刚被互联网公司裁员，HR拿赔偿金威胁他签自愿离职，他出来的时候心梗发作栽倒在斑马线，睁眼就看到手机弹出一个陌生APP推送：【幽冥便民服务中心临时办事员招聘，要求：熟悉互联网运营，会做KPI，包编制，底薪十万阴德，投递从速，过时魂飞魄散】
2. **剧情推进**：林默发现自己真的死了，周围飘着几个白无常，说最近阳间死人太多，地府编制不够，抓壮丁抓了他这个刚死的运营，正好匹配APP的职位要求；林默想拒绝，白无常说拒绝直接打落十八层地狱永世不得超生，只能硬着头皮接了入职邀请，绑定了「幽冥办公APP」；第一个自动推送的任务弹了出来：【一星委托：饿死鬼张栓的申诉处理，要求：解决张栓的投诉，任务时限：两个小时】
3. **爽点爆发**：林默在阳间天天处理客诉，本来就熟，直接按照阳间客诉处理流程梳理问题，见到饿死鬼张栓，三两句就问出了核心诉求，根本不像之前的地府办事员只会打官腔；
4. **结尾悬念**：张栓抹抹嘴说，我的核心诉求其实不是吃的，我是被阳间网红吃播活活气死的，我要他……话没说完，远处传来锁链哐当声，一个黑无常远远喊：林默？谁让你碰申诉鬼魂的，不知道这是老鬼们的自留地吗？
---

### 第2章：KPI拆解，饿死鬼的五星好评
1. **开篇钩子**：黑无常冲过来就要锁林默，说抢了本地鬼差的业绩要把他扔去恶鬼牢，林默直接打开幽冥APP甩规则：公开委托谁都能接，我完成拿积分，犯哪条规矩了？
2. **剧情推进**：黑无常被怼得说不出话，只能放话等着看林默完不成任务魂飞魄散；林默回头继续跟张栓聊，搞清楚了来龙去脉：张栓几十年前饿死，每年都能领地府的救济餐，最近地府搞改革，救济餐外包给了饿鬼道餐饮公司，饭越来越少，还天天让写好评，不写就停餐，张栓写了差评，被停了半个月餐，饿的受不了才来申诉，之前所有办事员都把他打发走了；
3. **爽点爆发**：林默直接用用户分层KPI拆解，先给张栓申请了临时救济餐饱餐一顿，然后找饿鬼道餐饮公司，拿出张栓的差评数据，说你们现在差评率15%，再涨上去地府采购就要扣你们的返点，不如给我个面子，给张栓恢复供应，我帮你们梳理差评原因优化产品；一顿操作下来，张栓满意，商家也点头，张栓直接改了五星好评，任务完成，APP到账10积分；
4. **结尾悬念**：林默刚拿到积分，手机APP突然弹出顶级推送：【紧急二星委托：奈何桥聂娇娇申请限流方案设计，酬劳50积分，点名要林默接，请问是否接单？】
---
### 第3章：积分兑换，这也能换还阳机会？
1. **开篇钩子**：林默愣了，自己刚完成一个小委托，怎么奈何桥的孟婆聂娇娇会点名找自己？黑无常在旁边看傻了，喃喃说聂娇娇那个姑奶奶的问题，地府没人敢接，你小子这下完了；
2. **剧情推进**：林默点开委托详情，原来阳间近百年人口爆炸，每天过奈何桥的鬼魂比千年之前多了一百倍，原来的一座奈何桥根本不够用，天天堵十几个时辰，孟婆天天加班累的要辞职，已经上访找阎王三百年了，阎王派了好几批人解决都没用，正好听说林默用现代方法解决了客诉，聂娇娇就找上门来了；林默打开APP的积分商城，意外发现商城最顶端赫然挂着：【还阳机会：积分100000，可兑换，保持全部记忆回到阳间】，旁边还有各种逆天道具：【阳间财运增幅：1000积分，【刀枪不入护身：5000积分】；
3. **爽点爆发**：林默算出解决奈何桥限流要是做成了，至少能拿几百积分，距离目标更近一步，直接接了委托，当场就给聂娇娇出了第一个思路：限流啊，这不简单，分时段预约啊，原来你们是随到随走当然堵，搞个线上预约系统不行吗？聂娇娇当场愣了，活了几千年从来没想过这个法子；
4. **结尾悬念**：林默刚说完，远处传来沉重的脚步声，一个穿黑龙袍的大胖子远远走过来，哈哈大笑：好一个分时段预约，林默是吗？跟我走，本王有个更大的项目，想要交给你……
