# 隐私政策

## Why have this document？

这项目目前没啥用户，也没啥隐私可泄漏的。但该写的还是写一下。

## 特别声明

本项目两位开发者皆为良善，不会进行网络攻击。

## 我们收集什么

- **账号信息**: 注册时的用户名、邮箱、密码（加密存储）
- **创作内容**: 你写的小说、设定、大纲——这些存在你自己的服务器上
- **使用数据**: 基本的使用日志（页面访问、功能使用频率）

## 我们不做什么

- 不卖你的数据
- 不会偷看你写的东西
- 不接第三方广告

## 数据存储

你的所有数据都存储在你部署的服务器上（PostgreSQL + ChromaDB），我们（或者说这个项目）没有云服务，数据不经过第三方。

## 联系

有问题直接在 GitHub 提 Issue。
