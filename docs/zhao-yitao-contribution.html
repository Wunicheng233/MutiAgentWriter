02 作家风格 Skill 注入层
让 AI 写的文字拥有"人味儿"——选择鲁迅、刘慈欣、海明威等名家风格，Writer 生成时自动注入对应文风。
backend/core/skill_runtime/
Core · Python
负责作家风格 Skill 的注入逻辑。用户在创建或管理项目时选择一个主作家风格，系统将其注入 Planner / Writer / Revise 等 Agent 的生成上下文。技能包含叙事视角、句式特征、修辞偏好、节奏控制、对话风格等多维信息，使生成文本保持风格一致性和个性化表达。

# Skill 注入 — Writer 生成前注入选定的作家风格 def apply_author_style(self, prompt: str, style: AuthorStyle) -> str: style_prompt = f"""当前采用 {style.name} 的文风进行创作： - 叙事视角：{style.perspective} - 句式特征：{style.sentence_pattern} - 修辞偏好：{style.rhetoric} - 节奏控制：{style.rhythm} - 对话风格：{style.dialogue} """ return style_prompt + "\n" + prompt
