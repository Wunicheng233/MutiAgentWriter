# 关于 StoryForge AI

## 我们做什么

StoryForge AI 是一个多智能体协作小说创作系统。

四个 AI Agent（Planner → Writer → Critic → Revise）像一支专业创作团队一样协作，帮你把故事构思变成完整的作品。你把握方向，AI 负责执行，不满意就修改——就像和一个真正的编辑团队合作。

在初赛中，我们迭代出了v2和v3版本，v2版本主要强化了infra，着重于强化写小说流程中的各个Agent组件，并加入了作家skill功能，可以让模型在生成文本时实现一些著名作家的文风（这个想法源于高中时一个喜欢用鲁迅体写小说的人🤫），并优化了前端的显示，带来全方位更佳的体验；v3版本主要加入了一个自我提升的loop，实现了更强大、具体的critic，并将

## 为什么做这个

~~因为我们无聊时喜欢读网文，但是社区的优质内容越来越少（不是）~~。

所以我们打算自己写小说啦！但是AI长篇小说写作有个核心痛点：**连贯性**，传统 AI 写作工具每次生成都是独立会话，不记得前文写了什么，写到后面忘了前面，所以我们打算实现一个写小说全方位碾压其他AI产品的**完整系统**，有时正巧赶上了新技术的出现，我们也乐于把他们接进来“试试”，尝试总归比等待好，我们觉得。

## 技术栈

- **前端**: React 19 + TypeScript + Vite + Tailwind CSS
- **后端**: FastAPI + SQLAlchemy + Celery
- **数据库**: PostgreSQL + Redis + ChromaDB
- **AI 模型**: 火山引擎、chatGPT-5.4-codex 、deepseek-v4-pro[1M]
