# 《夜语者》设定与大纲

## 一、核心设定
- **书名备选**：《夜语者》《第七层梦境》《无声证言》
- **一句话卖点**：一个能听见“物体记忆”的失语症患者，被迫用亡妻留下的旧物，找回她被掩盖的死亡真相。
- **时代背景**：当代都市，架空城市“临渊市”。
- **[时代校准]**：智能手机普及，全城监控网络完善，但地下室/老城区仍存在大量监控死角；社交媒体发达，但信息真伪混杂。

## 二、人物档案
### 主角：陆眠
- **身份**：前犯罪心理侧写师，现为独立旧物鉴定师。因一场事故患上“创伤性失语症”，无法说话。
- **性格**：极度敏感、观察入微、沉默寡言。
- **核心目标**：查明妻子苏晚的死亡真相，完成她未说完的遗言。
- **金手指/能力**：
  - **效果**：触摸特定旧物（尤其是亡者遗物），能“听见”该物体最近一次经历的强烈情绪片段与关键声音，宛如身临其境。能力触发时，眼球会短暂变为全黑。
  - **限制**：1. 只能听见“情绪峰值”前后的声音，信息残缺且无序。2. 每次使用会剧烈消耗精神，导致剧烈偏头痛与短暂耳鸣。3. 对金属、塑料等工业化无生命感的物体无效。4. 无法“看见”影像，只能听见声音。

### 关键配角（不超过3人）
| 姓名 | 身份 | 与主角关系 | 功能/性格 |
| :--- | :--- | :--- | :--- |
| 陈默 | 临渊市刑侦支队队长 | 陆眠的前同事、唯一信任的朋友 | 提供警方视角与资源，性格耿直，对陆眠的能力半信半疑，但选择无条件帮助。 |
| 周也 | 苏晚的生前闺蜜，心理医生 | 信息源与潜在威胁 | 表面温柔体贴，为陆眠提供心理支持，实则隐瞒着苏晚死前最后见面的关键信息。 |

## 三、世界规则
- **基础规则**：现实世界，无超自然力量。陆眠的能力被设定为“因脑部创伤导致的感官联觉异常”。
- **通用限制**：无。

## 四、分章大纲

**重要规则：每章目标字数是 `2000` 字，请根据这个字数规划本章情节容量。**

| 章节 | 本章目标（情节推进） | 核心冲突/爽点 | 结尾钩子 |
| :--- | :--- | :--- | :--- |
| 第1章 | 陆眠在整理亡妻苏晚遗物时，触摸到她的手机，意外触发能力，听到了苏晚死亡前最后一段通话的碎片：“……别告诉他，那个地下室……” | 能力首次展示的惊奇感 + 核心悬念抛出。 | 陆眠睁开眼，发现手机上显示的通话记录，正是苏晚死前打给他的，但他从未接到。而通话时长，是3分27秒。 |
| 第2章 | 陆眠找到陈默，试图通过警方记录还原苏晚最后48小时行程。陈默告知，苏晚死因是“意外坠楼”，现场无他杀痕迹，但有一张被撕碎的字条。 | 官方结论与主角感知的矛盾冲突。 | 陆眠在陈默离开后，触碰了那张被修复的字条碎片，听见一个陌生男人的低吼：“你不该查这个！” |
| 第3章 | 陆眠根据声音线索，前往苏晚常去的旧书店“墨香阁”，遇到店主，也是苏晚的闺蜜周也。周也表现出过分的关心与悲伤。 | 周也的言行产生疑点，信息差开始显现。 | 陆眠在书架角落发现一本苏晚借阅过的《地下室手记》，书页间夹着一张老照片，照片上是年轻时的苏晚与一个被涂黑脸的男孩。 |
| 第4章 | 陆眠通过能力“听”照片，得知照片里的男孩是临渊市十五年前一桩未结悬案“雨夜连环杀人案”的唯一幸存者。苏晚死前曾频繁调查此案。 | 案件线索升级，从个人悲剧转向陈年旧案。 | 周也打来电话，声音颤抖：“陆眠，别查了……苏晚她……是为了保护你。” |
| 第5章 | 陆眠调查雨夜案资料，发现凶手特征与陈默的一位线人高度吻合。陈默拒绝陆眠接触该线人，两人爆发激烈争执。 | 主角与权威（警方）的对抗，信任危机。 | 当晚，陆眠收到一个匿名包裹，里面是苏晚的日记本，但关键几页被撕走。日记最后一句话是：“我见到了‘他’，他就在我们身边。” |
| 第6章 | 陆眠利用能力“听”日记本，听到苏晚和一个人在咖啡馆争吵的声音。声音经过变声处理，无法辨认。争吵中提到“名单”和“证据”。 | 迷雾加重，幕后黑手浮现。 | 陆眠发现日记本封皮夹层里有一枚微型U盘，插入电脑后，屏幕上出现一个倒计时：72:00:00。 |
| 第7章 | 倒计时迫使陆眠加快行动。他利用侧写技能，结合U盘里的加密文件碎片，推断出雨夜案真凶可能是一位有权势的政客。陈默被上级施压，要求停止调查。 | 主角孤军奋战的爽点+政治黑幕的压迫感。 | 陆眠跟踪周也，发现她在深夜进入一栋废弃大楼，正是苏晚手机里提到的“地下室”。 |
| 第8章 | 陆眠潜入地下室，发现这里是一个被改造的审讯室。他“听”到了苏晚在此处被逼问的录音碎片，得知苏晚掌握了政客与十五年前谋杀案有关的直接证据。 | 真相的逐步揭露，情绪爆发点。 | 陆眠被躲在暗处的人击晕。黑暗中，他听到一个熟悉的声音：“你和她一样，太聪明了……但聪明人活不长。” |
| 第9章 | 陆眠醒来，发现自己被绑在地下室。面前的人是周也。周也露出真面目，她一直是政客的情人和帮凶，接近苏晚是为了监视。 | 身份反转的震撼+主角陷入绝境。 | 周也拿出注射器，准备杀人灭口。此时，陈默带人破门而入。 |
| 第10章 | 陈默与周也对峙。周也试图用政客的力量压人。陆眠在混乱中挣脱束缚，触碰了周也的手机，听到了她与政客的通话录音，作为证据。 | 高潮对决，正义与邪恶的正面交锋。 | 政客被捕的消息传来。但在警车上，陆眠看到周也露出一个诡异的微笑。 |
| 第11章 | 周也在审讯中全盘招供，但将所有罪行推给政客。她声称自己是被胁迫。案件看似告破。陆眠却觉得哪里不对。 | 看似真相大白的假象。 | 陆眠再次触碰苏晚的手机，这次他听到了苏晚死前最后一句完整的话：“周也……你才是……那个男孩……” |
| 第12章 | 陆眠意识到，周也就是十五年前那个幸存下来的“男孩”。她为了掩盖身份，长大后整容变性，并设计杀死了唯一知道真相的苏晚。 | 最终反转，真相大白。 | 陆眠在法庭上提交了新证据。周也被定罪。最后一幕，陆眠站在苏晚墓前，将一枚戒指放在墓碑上。他轻轻地，无声地说出了一句话。字幕浮现：【陆眠终于找到了最后一个“答案”】。 |
