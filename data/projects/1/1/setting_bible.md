# 《重光》设定圣经
---
## 一、世界观底层规则与时代背景
### 1.时代基本设定
- **时间线：** 故事起始于1328年，终结于1368年洪武元年朱元璋称帝，完全遵循元末明初正史时间框架
- **科技存在规则：**
  ✅ 1328年故事起始时已存在/已普及的技术：水车、牛耕、活字印刷、内河成熟帆船造船术、元军少量装备未普及的火铳
  ❌ 1328-1368年故事全程绝对不能出现的技术：蒸汽机、现代炼钢、近代火器战术等超出明初生产力的超前科技
- **整体社会背景：** 元廷政治腐败，皇位更迭频繁，黄河泛滥灾荒连年，汉人税负沉重，白莲教弥勒教地下传播，流民遍地，四等人制下南人地位最低，底层汉人几乎无上升空间，民族矛盾与阶级矛盾总爆发，时代底层逻辑为**「活下去就是最大的正义」**，所有人物选择都围绕生存与权力展开，无天生圣人和天生恶魔，所有性格选择都有底层逻辑支撑。

### 2.核心制度设定
- 官制：元末沿用蒙古行省-路-府-县四级官制，达鲁花赤掌实权，汉人多只能任佐官；红巾军沿用部分宋制，朱元璋逐步恢复汉官制
- 军制：元廷实行军户制，后期战斗力低下；红巾军为募兵制，朱元璋采用「屯田养兵」制度，建立纪律严明的野战部队

## 二、完整人物人设（不可修改）
### 1.核心主角：朱元璋（朱重八）
| 项目 | 设定内容 |
| ---- | ---- |
| 身份变迁 | 濠州钟离佃户之子→皇觉寺行童→红巾军将领→吴国公→大明开国皇帝 |
| 外貌相关特殊印记 | 随身携带皇觉寺度牒与马秀英定情玉佩，终身不忘饥饿滋味，晚年仍常吃粗麦饭 |
| 性格 | 极度自尊与自卑交织，隐忍多疑，冷酷果决，对底层苦难有刻入骨髓的记忆，对背叛零容忍；对马秀英保留一生柔软，对兄弟从信任到逐步猜忌，有欲望但不滥情，行事务实不尚虚文 |
| 核心目标变迁 | 初始只求活下去→中期想要给天下百姓一个吃饱饭的世道→最终变成守住朱家天下的孤独帝王 |

### 2.核心女主：马秀英（马皇后）
| 项目 | 设定内容 |
| ---- | ---- |
| 身份变迁 | 郭子兴养女→朱元璋妻子→大明开国皇后 |
| 性格 | 外柔内刚，通透聪慧，有普通人的善良，也有惊人的政治远见，是朱元璋黑暗岁月里唯一的缓冲，永远能在朱元璋狂躁的时候让他冷静 |
| 核心目标 | 陪朱元璋活下去，守住这个家，在朱元璋滥杀时尽可能保全无辜 |
| 能力 | 善于收拢人心，在朱元璋被郭子兴猜忌时靠自身周旋化解危机，是朱元璋事业最早的合伙人 |

| ---- | ---- | ---- | ---- |

| 刘伯温 | 元廷进士→朱元璋谋臣 | 清高多智，精通谋略，不结党 | 朱元璋的战略引路人，代表浙东文官集团，是帝王权术的平衡点 |

## 三、人物关系网
1. 朱元璋与马秀英：夫妻，马秀英是朱元璋一生唯一的柔软缓冲，也是最早的事业合伙人
2. 朱元璋与徐达、汤和：同乡创业兄弟，朱元璋对其从绝对信任逐步转为猜忌，汤和主动放权得以善终，徐达始终谨慎
3. 朱元璋与李善长、刘伯温：君臣，李善长为淮西领袖，刘伯温为浙东领袖，是朱元璋平衡朝局的两枚棋子
4. 朱元璋与郭子兴：翁婿，郭子兴是朱元璋最早的起家依靠，郭子兴死后朱元璋接收其部众
5. 朱元璋与陈友谅、张士诚：敌对割据势力，陈友谅是主要对手，张士诚是中期牵制对手
6. 朱元璋与朱文正：叔侄，朱文正是朱元璋侄子，洪都保卫战功臣，后因谋反被废

## 四、全书时间线大纲（完全对应正史）
1. **第一卷 濠梁孤影：1328年-1352年** 共20章：濠州饥荒，朱重八失去家人，入皇觉寺为僧，流浪三年归寺，接到汤和投军密信，杀僧（放僧）跑路投奔郭子兴，逐步在红巾军站稳脚跟，最后渡江攻克采石，目标直指集庆
2. **第二卷 剑起淮西：1352年-1360年** 共20章：朱元璋攻克集庆改应天建立根据地，三请刘伯温定下战略，平定邵荣叛乱，龙湾大败陈友谅，攻取江西，对峙陈友谅，彻底坐稳江南地盘
3. **第三卷 龙战于野：1360年-1367年** 共20章：鄱阳湖决战击杀陈友谅，平定谢再兴叛乱、处理朱文正谋反，围攻平江攻灭张士诚，统一江南
4. **第四卷 日月重开：1367年-1368年** 共20章：发布北伐檄文，徐达常遇春北伐，平定山东河南，攻克大都收复燕云十六州，击败王保保，朱元璋应天称帝，全书以朱元璋深夜独自对山河的孤独背影结束

## 五、创作规则要求
1. 所有重大事件、官制军制、社会细节严格遵循元末明初史实，拒绝过度戏说与降智爽文设定
2. 严格遵循节奏规则：每3章一个小高潮，每章结尾必须留悬念钩子，开篇300字内抛出核心冲突
3. 人物塑造坚持复杂立体原则，拒绝脸谱化，所有人物行为必须符合本设定的人设与时代逻辑
